Bloggs' Bar


This script willl copy the first TextShadow from array, delete it and then add the first TextShadow to end of array. Pass in a value of milliseconds to set the frequency of cycling through the array of textshadows.

As an example, you have 3 TextShadows; red, orange, yellow. The sequence will go like this.
First Cycle: orange, yellow, red
Second Cycle: yellow, red, orange
Third Cycle: red, orange, yellow

It will keep doing this until you've stopped the timer.
