Neon Builder.JS


Basic Example - A simple script which creates a textShadow by adding top/left dimensions, blurring values and a colour value.

Update Example 1 - A script which updates a textShadow in the array by index. The array of textShadow effects are then applied to the script. This script shows you how to update all of textshadows in the array.

Update Example 2 - A script which updates a textShadow in the array by index. This example shows you how to apply blurring value to a textShadow

Update Example 3 - A script which updates a textShadow in the array by index. This example shows you how to set your textshadow left/top position.

Delete Example - A script which removes a textShadow from the array by index. This example shows you how to do this.
